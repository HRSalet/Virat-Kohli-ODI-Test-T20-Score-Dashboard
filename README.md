# Virat Kohli ODI T20 Performance Dashboard

## 1. Overview

This Power BI dashboard provides a comprehensive analytical view of Virat Kohli's ODI, Test, and T20 cricket performance across multiple dimensions including temporal trends, venue-based analysis, and opponent-specific statistics. The report enables stakeholders to track performance metrics, identify patterns in scoring, and analyze career trajectory through interactive visualizations that support data-driven decision making for team strategy and performance evaluation.

## 2. Purpose

This dashboard was designed to serve mutiple stakeholders with distinct analytical needs:<br>

- Identify venue-specific performance patterns to optimize team selection and batting order.
- Track consistency metrics across seasons to evaluate career progression.
- Understanding scoring patterns against specific teams to develop targeted game plans.
- Access historical data to make informed predictions for upcoming matches.
- Understand which grounds yield the highest scoring potential.

## 3. Tech Stack<br>

The dashboard was built using the following tools and technologies:
- 📊 Power BI Desktop – Main data visualization platform used for report creation.
- 📂 Power Query - Data Transformation and cleaning layer for reshaping and preparing the data.
- 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, and dynamic visuals.
- 📁 File Format – .pbix for development and .png for dashboard previews.

## 5. Key Metrics

**Total Runs:** The metric shows that Virat Kohli has achieved a significant total of 23709 runs from ODI, Test, and T20 formats.

**Maximum Runs:** The Virat Kohli scores highest 254 runs in a single match.

**Average Run Rate:** This metric shows that overall average run rate of the Virat Kohli is 45.95.

**Opponent's Matches:** The Virat Kohli played 516 matches against opponent teams in ODI, Test, and T20 formats.

These KPIs are displayed to give users instant insights into the Virat Kohli overall performance, serving as a quick reference for stakeholders and decision-makers.

## 6. Visualizations

The dashboard utilizes different types of visualizations to represent data in an easily understandable format, allows for deeper insights into the Virat Kohli cricket performance.

**1. Runs by Top 5 Ground (Bar Chart)**<br>
This visualization displays Virat Kohli's top performances in different grounds. This visual gives insights into Virat Kohli's favorite cricket ground. For e.g. at Wankhede cricket stadium, Virat Kohli scores the highest runs in ODI, Test, and T20 formats.

**2. Top 5 Performance by Ground (Multi-Row Card)**<br>
This visualization represents the top highest scoring ground by runs. It also reflects Virat Kohli's performance against opponent teams in the ground.

**3. Yearly, Quarterly, and Monthly Runs (Table)**<br>
This table offers an overview of the Virat Kohli yearly, quarterly and monthly runs. Also shows the total runs scored by Virat Kohli. It displays every year with associated runs, enabling understandable analysis and comparison.

**4. Highest Runs Vs. Opponent Team (Scroller)**<br>
This is a scroller visual that displays the highest runs scored by Virat Kohli against opponent teams. This gives an insight about how Virat Kohli faces challenges against these opponent teams and is useful for predicting scores in upcoming matches.

## 7. Filters

The dashboard features interactive components to improve user experience and data analysis. At the top of the dashboard, year filters from 2008 to 2022 are present, which indicates that users can flexibly change years shown to concentrate on particular intervals. Similarly, at the bottom, a date filter is there for checking out Virat Kohli's score in detail. This feature is essential for analyzing history, recognizing trends, and planning for the future.
